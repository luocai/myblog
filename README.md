# myblog
* 这个一个基于ssm框架搭建的个人博客，目前已经部署到阿里云服务器
* 在线演示：<a href="http://119.23.63.3:8080/myblog/index">诸葛财财的个人博客</a>

# 如何使用
* 1 clone下来后运行创建名为myblog的数据库，并运行sql文件，修改jdbc.properties为自己的配置
* 2 修改七牛云配置为自己的配置
* 3 如遇到问题可联系我。。。
